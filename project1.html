3D Rotating Design with Interactive Animation

This project showcases a visually appealing 3D rotating design created using pure HTML, CSS, and a touch of JavaScript for interactivity. The design features six concentric squares, each dynamically rotating at different speeds, creating an engaging 3D effect.

Key Features:

1) Smooth Animations:

A custom @keyframes animation gives each square a unique rotation effect, simulating a mesmerizing 3D motion.

2) Interactive Hover Effects:

When hovered over, each square pauses its rotation and changes color, creating an engaging user interaction.

3) Click-Based Interactions:

Users can click on any square to change its color dynamically, demonstrating JavaScript-driven interactivity.

4) Responsive Center Alignment:

The design is centrally positioned on the screen using CSS flexbox techniques, ensuring it looks visually balanced across various screen sizes.

5) Modern Aesthetic:

A soothing blue palette combined with shadows creates a modern and elegant 3D visual appeal.

Applications:

This design can be adapted for web banners, loading animations, or interactive web elements to enhance user experience.
